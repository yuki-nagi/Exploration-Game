# Exploration-Game
日本語の探索ゲームです
※コマンドは英語です。

気がつけば密室に1人で囚われていた探索者のあなた。
あなたは部屋を探索し、この部屋から脱出しなければなりません。
扉の鍵を探すも良し、他の出口を探しても構いません。

コマンド一覧
look + "対象"  ……対象をちゃんとよく見ます。本の場合は読みます。
                   対象を指定しない場合、部屋を見回します。

mebosi + "対象" ……対象に対して脱出にかかわるヒントがないか探ります。

break + "対象" ……対象が壊せるものなら壊します。

help ……現在使えるコマンド一覧を表示します。
          新たにコマンドが使えるようになった場合、helpに追加されます。
          
          
例：look Memo →メモを見ます
